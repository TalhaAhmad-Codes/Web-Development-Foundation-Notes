# HTML Headings (`<h1>` – `<h6>`) and Hierarchy

HTML headings define the **structure and outline** of a web page. They help browsers, search engines, and screen readers understand **content importance**.

---

## Heading Tags Overview

HTML provides **six heading levels**:

```html
<h1> – Main heading
<h2> – Section heading
<h3> – Sub-section heading
<h4> – Sub-sub-section
<h5> – Minor heading
<h6> – Least important heading
```

---

## Basic Example

```html
<h1>Web Development</h1>
<h2>HTML Basics</h2>
<h3>Headings</h3>
```

This represents a **logical content hierarchy**.

---

## Heading Hierarchy Rules

### 1. Use `<h1>` Once per Page

* Represents the **main topic**
* Important for SEO and accessibility

```html
<h1>DevNotes - HTML Guide</h1>
```

---

### 2. Do Not Skip Levels

❌ Incorrect:

```html
<h1>Main</h1>
<h3>Skipped Level</h3>
```

✔ Correct:

```html
<h1>Main</h1>
<h2>Section</h2>
<h3>Sub-section</h3>
```

---

### 3. Use Headings for Structure, Not Styling

❌ Incorrect:

```html
<h4>Text just for size</h4>
```

✔ Correct:

```html
<p class="large-text">Text</p>
```

Styling should be handled by **CSS**, not heading level.

---

## Visual Hierarchy (Concept)

```
<h1> Page Title
 ├─ <h2> Section
 │   ├─ <h3> Sub-section
 │   │   └─ <h4> Details
```

---

## Accessibility Benefits

* Screen readers navigate using headings
* Users can jump between sections
* Improves usability for all users

---

## SEO Benefits

* Search engines use headings to understand content
* `<h1>` and `<h2>` carry higher importance
* Clean hierarchy improves ranking

---

## Headings in Real Projects

### In ASP.NET / Razor

```html
<h1>@Model.Title</h1>
```

### In Blazor / React

```html
<h2>Product Details</h2>
```

The rules remain **exactly the same**.

---

## Best Practices Summary

✔ One `<h1>` per page
✔ Follow order (`h1 → h2 → h3`)
✔ Use headings for meaning, not size
✔ Style with CSS

---

## Quick Summary

| Tag           | Purpose              |
| ------------- | -------------------- |
| `<h1>`        | Page title           |
| `<h2>`        | Main section         |
| `<h3>`        | Sub-section          |
| `<h4>`–`<h6>` | Lower-level headings |

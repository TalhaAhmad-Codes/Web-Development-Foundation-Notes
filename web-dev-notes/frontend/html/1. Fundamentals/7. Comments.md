# Comments in HTML

HTML comments are used to **add notes or explanations** inside HTML code that are **ignored by the browser**.

---

## Syntax

```html
<!-- This is an HTML comment -->
```

---

## Example

```html
<!-- Header Section -->
<header>
    <h1>Welcome</h1>
</header>
```

---

## Multi-line Comment

```html
<!--
This is a multi-line
HTML comment
-->
```

---

## Common Uses

* Explaining code sections
* Temporarily disabling HTML
* Adding notes for developers
* Improving readability

---

## Important Notes

* Comments are **not displayed** on the page
* Comments are **visible in page source**
* Avoid putting sensitive data in comments

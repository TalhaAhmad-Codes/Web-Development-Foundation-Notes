# Basic Structure of an HTML Document

Every HTML page follows a **standard structure** that helps browsers understand **how to parse and render content** correctly.

---

## Minimal HTML Document

```html
<!DOCTYPE html>
<html>
<head>
    <title>My Web Page</title>
</head>
<body>
    <h1>Hello World</h1>
    <p>This is a basic HTML document.</p>
</body>
</html>
```

This structure is **mandatory** for all modern websites.

---

## `<!DOCTYPE html>`

### What It Is

* Declares the **HTML version** used in the document
* Ensures the browser uses **standards mode**
* Required at the very top of the file

### Why It Matters

Without it, browsers may enter **quirks mode**, causing:

* Layout issues
* CSS inconsistencies
* Unexpected rendering behavior

### Example

```html
<!DOCTYPE html>
```

## `<html>` Element

### What It Is

* The **root element** of an HTML document
* Wraps all content on the page

### Responsibilities

* Contains `<head>` and `<body>`
* Defines the document as HTML

### Example

```html
<html>
    <!-- Entire document lives here -->
</html>
```

### Best Practice (Language Attribute)

```html
<html lang="en">
```

This helps:

* Screen readers
* Search engines
* Accessibility tools

---

## `<head>` Element

### What It Is

The `<head>` contains **metadata**, not visible page content.

### What Goes Inside `<head>`

* Page title
* Character encoding
* SEO metadata
* CSS links
* JavaScript references (in some cases)

### Common `<head>` Example

```html
<head>
    <meta charset="UTF-8">
    <title>DevNotes - HTML Basics</title>
</head>
```

ğŸ§  **Key Point:**
Anything inside `<head>` **does not appear on the page**.

---

## `<body>` Element

### What It Is

The `<body>` contains **all visible content** of the web page.

### What Goes Inside `<body>`

* Headings
* Paragraphs
* Images
* Buttons
* Forms
* Links

### Example

```html
<body>
    <h1>Welcome</h1>
    <p>This content is visible to users.</p>
</body>
```

ğŸ§  **Key Point:**
Only one `<body>` element is allowed per document.

## Key Takeaways

* HTML documents follow a **strict structure**
* `<!DOCTYPE html>` ensures correct rendering
* `<html>` is the root container
* `<head>` stores metadata
* `<body>` holds visible content
* This structure applies to **ASP.NET, Razor, Blazor, React, Angular**

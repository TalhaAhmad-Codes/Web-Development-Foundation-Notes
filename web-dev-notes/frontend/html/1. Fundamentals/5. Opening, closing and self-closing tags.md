# Opening, Closing & Self-Closing HTML Tags

HTML uses **tags** to define elements. These tags are categorized based on **how they wrap or represent content**.

---

## Opening Tags

### Definition

An **opening tag** marks the **start of an HTML element**.

### Syntax

```html
<tagname>
```

### Example

```html
<p>
<h1>
<div>
```

Opening tags may also contain **attributes**.

```html
<a href="https://example.com">
```

üß† **Key Point:**
Opening tags **do not close themselves** (unless they are self-closing elements).

---

## Closing Tags

### Definition

A **closing tag** marks the **end of an HTML element**.

### Syntax

```html
</tagname>
```

### Example

```html
</p>
</h1>
</div>
```

Closing tags:

* Always start with `/`
* Never contain attributes

üß† **Key Point:**
Most HTML elements **require a closing tag**.

---

## Elements with Opening & Closing Tags

These elements **wrap content**.

### Example

```html
<p>This is a paragraph</p>
```

```html
<h1>Main Heading</h1>
```

```html
<div>
    <p>Nested content</p>
</div>
```

Common elements requiring closing tags:

* `<p>`
* `<h1>` ‚Äì `<h6>`
* `<div>`
* `<section>`
* `<article>`
* `<span>`

---

## Self-Closing Tags

### Definition

**Self-closing tags** represent elements that **do not wrap content**.

* No closing tag required
* Content is defined using attributes

### Syntax (HTML5)

```html
<tagname>
```

### Alternative Syntax (XHTML / XML-style)

```html
<tagname />
```

Both are valid in HTML5.

---

## Common Self-Closing Elements

```html
<img src="logo.png" alt="Logo">
<br>
<hr>
<input type="text">
<meta charset="UTF-8">
<link rel="stylesheet" href="styles.css">
```

üß† **Important:**
Self-closing elements **cannot contain child elements**.
